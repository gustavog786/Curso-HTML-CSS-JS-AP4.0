Notas sobre la estructura

  • para que aparezca toda la estructura armada podemos poner el ! o html:5
  • lo segundo a configurar son los espacios, abajo a la derecha, pasarlo del predeterminado en 4 a 2
  • debemos poner el idioma en el que se desarrollará la pagina, lang es el atributo y "es" es el valor

head

  • dentro de head esta todo lo no visible de mi pagina, es decir la configuración
  • title es el nombre de la pestaña

body

  • dentro del body esta todo lo visible de mi pagina
  • aqui podemos poner etiquetas para titulos y subtitulos con h1 hasta h6, p para el texto, lorem para agregar texto de relleno

!DOCTYPE

Todos los documentos HTML deben comenzar con una declaración <!DOCTYPE>. La declaración no es una etiqueta HTML. Es una "información" para el navegador sobre qué tipo de documento esperar. En HTML 5, la declaración es simplemente: <!DOCTYPE html>

Etiquetas de texto de HTML

  • main: es el contenido principal dentro del body
  • h1-h6: titulos
  • p: parrafo
  • br: para saltos de linea (no tiene cierre)
  • img: imagen (esta etiqueta no se cierra)
  • a: ancla para luego redireccionar dentro de la misma pagina
  • ul: listado desordenado
  • ol: listado ordenado
  • hr: linea horizontal
  • comentarios !--
  • strong: negrita
  • em: cursiva
  • s: tachado
